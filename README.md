Problem 1: File Reader & Writer
Task:
Create a Node.js script that:
1. Reads a file input.txt
2. Counts:
○ Total words
○ Total lines
3. Writes result into output.txt
Input Example (input.txt)
Hello students
This is Node.js exam

Output (output.txt)
Total Lines: 2
Total Words: 6

Requirements
● Use fs module (fs.readFile, fs.writeFile)
● Use async or callback (any one is fine)
● Handle file not found error
